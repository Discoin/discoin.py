# Discoin for Python

This is the official discoin library for python. Here is discoin's [Gitbook](https://discoin.gitbook.io/docs/)

## Requirements
* Python 3.6 or up
* aiohttp

## Installation
```bash
$ pip install discoin
```

Documentation can be found [here](https://discoinpy.readthedocs.io)
